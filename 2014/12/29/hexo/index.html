安装theme

我安装的是Pacman
cd 到刚才创建的目录下

然后theme文件夹下会多一个pacman的文件夹，修改_config.yml,theme: pacman。现在就有
两个_config.yml文件让我们修改了，一个在主文件下，一个在/themes/pacman/，高兴不。

安装rss

这个很简单
开启 atom 生成插件

使用 npm 下载 hexo-generator-feed 插件

修改/themes/pacman/下的_config.yml
rss: /atom.xml

安装sitemap

hexo/_config.yml
plugins:
- hexo-generator-sitemap
npm install hexo-generator-sitemap
