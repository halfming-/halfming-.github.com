Hugo常用命令

Jul 16, 2017

生成一个网站

hugo new site blog

应用一个theme

cd themes
git clone https://github.com/spf13/hyde.git

添加post

hugo new post/first.md

上传到github

hugo --theme=hyde --baseUrl="http://halfming.github.io/"

cd public
git init
git remote add origin https://github.com/halfming/halfming.github.io.git
git add -A
git commit -m "first commit"
git push -u origin master
